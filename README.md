Wiki for the Auckland University Gōjū Ryū Karate Club, written in Vue.

Not currently hosted as it is still under development. However, you can visit
the current wiki [here](https://wiki.uoa-karate.club)

## Running the site

#### Install

`yarn install`

#### Serve

`yarn serve`

#### Build

`yarn build`
